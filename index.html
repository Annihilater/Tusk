CKPX - Chrome KeePass, Extended

This extension adds KeePass autofill functionality to Google Chrome without compromising security for convenience.

CKPX is a fork of CKP, and seeks to provide improvements to the UI, better security practices, and KDBX version 4 support.

Under Active Development
Autofill or Copy KeePass Passwords
Google Drive, OneDrive, or Dropbox Support

A major issue with CKP is currently that your master password can be stored indefinitely ON DISK regardless of how long you set for the remember time. CKPX seeks to re-implement the master password remember feature to be entirely in-memory so that your password could not be recovered from a computer that is powered off.
