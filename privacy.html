CKP

Chrome extension for filling usernames & passwords stored in KeePass databases

View on GitHub Download .zip Download .tar.gz

Privacy Policy

  • CKP may transmit and receive user data between the user's own storage at one of the cloud storage providers (e.g. Google Drive, Dropbox) and CKP extension. When doing so, data is always encrypted using the mechanism supported by the cloud provider.
  • CKP does not transmit data to anywhere not mentioned above.
  • CKP only scans webpages when the user has opted-in to fill passwords on that particular website.
  • When CKP scans a webpage, it is for the sole purpose of identifying username and password fields, so that they may be automatically filled at the user's request.
